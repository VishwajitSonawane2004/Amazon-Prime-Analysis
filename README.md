# Amazon Prime Video Data Analysis

## 📌 Project Overview
This project analyzes Amazon Prime Video content data to extract insights about content diversity, trends over time, regional distribution, and ratings.

---

## 🎯 Problem Statement
The objective of this project is to analyze Amazon Prime Video content data to understand audience preferences and content strategies.

---

## 📂 Dataset Description
- Titles dataset: Contains information about movies and TV shows  
- Credits dataset: Contains actors and directors data  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📊 Key Insights
- Drama and Comedy are the most common genres  
- Content production has increased over the years  
- Popularity and ratings are not always correlated  
- The United States produces the majority of content  

---

## 📈 Visualizations
- Bar charts  
- Histogram  
- Heatmap  
- Scatter plot  

---

## 🏁 Conclusion
Amazon Prime has a diverse and growing content library. The analysis helps understand user preferences and supports better content strategy decisions.
